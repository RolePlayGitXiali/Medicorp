# Guide d'Utilisation - Médicorp

## Cas d'Usage Pratiques

### Cas 1 : Commander un kit complet d'implants T1

1. Ouvrez l'onglet **Implants**
2. Assurez-vous que le tier **T1** est sélectionné
3. Ajoutez 1 unité de chaque type :
   - Hyperdermique
   - Servomoteur
   - Biomoniteur
4. Dans l'onglet **Abonnement**, sélectionnez "Abonnement Premium"
5. Consultez le résumé pour voir les prix avec réduction (20%)

**Résultat attendu :**

- Prix de base : ~1430 €
- Réduction (20%) : ~286 €
- Prix final : ~1144 €
- Marge : ~432 €

---

### Cas 2 : Commander des stims T3 assortis

1. Ouvrez l'onglet **Stims**
2. Changez le tier en **T3**
3. Ajoutez 2 unités de chaque type :
   - Bota
   - Adrénaline
   - Endurol
   - Stéroide
4. Garder "Sans Abonnement" pour voir le prix plein
5. Consultez la marge bénéficiaire

**Résumé :**

- 8 produits × 2 unités = 16 articles
- Prix de base : ~3240 €
- Sans réduction
- Marge importante

---

### Cas 3 : Mélanger des catégories

1. **Onglet Implants** (T2) : Ajoutez 1 Servomoteur
2. **Onglet Éléments** : Ajoutez 5 Kits de Soins T3
3. **Onglet Stims** (T2) : Ajoutez 3 Bota
4. Sélectionnez "Abonnement Elite" (30%)
5. Vérifiez que seuls les produits réductibles reçoivent la réduction

**Points importants :**

- Grenade Kolto et Bota Cryo ne sont PAS réductibles
- Les Kits de Soins T2/T3 et Stims le sont
- La marge est plus intéressante avec les implants

---

### Cas 4 : Consulter les matériaux

1. Ouvrez l'onglet **Matériaux**
2. Consultez la liste complète avec prix/unité
3. Comparez les coûts matériau avec les prix de vente
4. Identifiez les matériaux onéreux (Platinum: 50€, Bacta: 60€)

## Astuces d'Utilisation

1. **Maximiser la réduction** : Commandez des produits réductibles avec l'Abonnement Elite
2. **Analyser la marge** : Les implants T2 MK2 offrent les meilleures marges
3. **Mélanger les tiers** : Vous pouvez ajouter des produits de tiers différents au même panier
4. **Vérifier les matériaux** : Consultez l'onglet Matériaux pour planifier l'approvisionnement
5. **Exporter les calculs** : Le résumé affiche tous les calculs détaillés

---

## Raccourcis Clavier

- **Onglets** : Cliquez sur les boutons d'onglets en haut
- **Tiers** : Cliquez sur T1, T2, T2 MK2, T3 selon l'onglet
- **Ajout rapide** : Entrez la quantité puis cliquez "+"
- **Suppression rapide** : Cliquez "✕" à côté d'un article du panier

---

## Dépannage

**Le panier ne change pas :**

- Vérifiez que le serveur est en cours d'exécution
- Rafraîchissez la page (F5)
- Vérifiez la console navigateur (F12) pour les erreurs

**Les réductions ne s'appliquent pas :**

- Vérifiez que le produit a "Réductible: ✅"
- Confirmez que l'abonnement a un % de réduction > 0

**Les quantités ne s'ajoutent pas :**

- Vérifiez que le nombre est > 0
- Essayez avec un nombre simple (1, 2, 3)
